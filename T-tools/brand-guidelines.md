# brand-guidelines

**Full skill:** `.claude/skills/brand-guidelines/SKILL.md`
**סוג:** Project skill

## מה הסקיל עושה
מכיל את ה-DNA של המוצר — visual language, Golden Rules, decision flow. כל עבודת UI, copy, עיצוב, צבע, פונט, אנימציה, או פיצ'ר חדש חייבת לעבור דרך הסקיל הזה לפני קידוד.

## מתי לטעון
- לפני כל שינוי UI/עיצוב
- לפני כתיבת copy (error messages, buttons, empty states)
- לפני בניית קומפוננט חדש
- לפני תכנון מסך חדש

## מי משתמש
- Claude (Orchestrator) — כל עבודת brand/design
- סוכן קוד (עתידי) — לפני כל UI implementation

## עקרונות עיקריים
- Vibe: ביטחון, עוצמה, זרימה
- Dark only | RTL default | Hebrew first
- Stack: Leaflet, Tailwind, lucide-react בלבד
- אדום = חירום בלבד | ירוק = פעיל/בטוח בלבד
