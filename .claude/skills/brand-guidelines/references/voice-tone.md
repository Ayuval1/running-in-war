# Voice & Tone

## Core Voice Principles
1. **ישיר** — אל תרבה במילים. כלל: אם אפשר בשתי מילים, אל תכתוב חמש.
2. **קצר** — משפט אחד עדיף על שניים. נקודה > פסיק.
3. **בהיר** — לא ז'רגון, לא slang, לא מטאפורות מסובכות.
4. **פרגמטי** — מה לעשות, איפה ללחוץ, כמה זה רחוק. בלי "happy talk".
5. **רגוע** — גם בחירום, הטון לא היסטרי. דחוף ≠ פאניקה.

## Tone by Context

### Emergency (אזעקה / SOS active)
**טון:** דחוף, פעמי, ציווי קצר.
**אורך:** 1-2 שורות מקסימום.
**Caps:** מותר באירוע חיים-מוות בלבד.

✅ **דוגמאות:**
```
אזעקה אזעקה אזעקה
היכנס למקלט הקרוב בעוד 80 מטרים
יש לך 30 שניות להגיע
```
```
אזעקה פעילה — לך למקלט עכשיו!
```
```
SOS — מצא מקלט קרוב
```

❌ **לא:**
- "🚨 שים לב — אזעקה באזורך!" (טון רך)
- "אנא היכנס למקלט הקרוב בהקדם האפשרי" (פורמלי)
- "אזעקה זוהתה במערכת" (טכני / מרוחק)

---

### Normal Operation (שגרה)
**טון:** יבש, פרגמטי, ענייני. כמו ציוד מקצועי שמדבר אליך.
**אורך:** משפט אחד.

✅ **דוגמאות:**
```
GPS לא זמין, וודא חיבור למיקום
```
```
3 מקלטים סומנו
```
```
מוכן לחלוטין!
```

❌ **לא:**
- "לא הצלחנו לאתר אותך 😞" (רגשני)
- "ברוך הבא! התחל לסמן מקלטים" (חברותי מדי)
- "המערכת זיהתה כי 3 מקלטים סומנו" (פורמלי)

---

### Empty States
**טון:** ענייני + הדרכה ברורה לפעולה הבאה.
**מבנה:** מצב (1 שורה) + מה לעשות (1 שורה) + כפתור פעולה.

✅ **דוגמה:**
```
אין מקלטים עדיין
לחץ על "מפה" וסמן את המקלטים הקרובים אליך
[פתח מפה]
```

✅ **דוגמה:**
```
אין מסלולים קיימים בחשבון זה
[תכנן מסלול חדש]
```

❌ **לא:**
- "אופס, נראה שאין כאן כלום עדיין!" (חמוד מדי)
- "List is empty" (אנגלית בלי סיבה)

---

### Errors
**טון:** עובדתי, מה קרה + מה לעשות.
**מבנה:** הבעיה (קצר) + פעולה לתיקון.

✅ **דוגמאות:**
```
GPS לא זמין, וודא חיבור למיקום
```
```
לא ניתן לטעון מקלטים — בדוק חיבור לאינטרנט
```
```
כתובת לא נמצאה
```
```
שגיאה בשמירה
```

❌ **לא:**
- "אופס! משהו השתבש 🤷" (לא מועיל, חמוד)
- "Error: GEOLOCATION_PERMISSION_DENIED" (טכני / קוד)
- "צר לנו, אך לא הצלחנו לבצע את הפעולה" (פורמלי / מנומס)

---

### Buttons (ציווי פעולה)
**כלל:** ציווי + פועל + (אובייקט אם צריך). מקסימום 2-3 מילים.

✅ **טוב:**
- `תכנן מסלול`
- `פתח מפה`
- `מצא מקלט קרוב`
- `שמור`
- `כניסה` / `הרשמה`
- `הוסף מקלט`
- `שתף מסלול`
- `מחק`

❌ **לא:**
- `לחץ כאן כדי לתכנן מסלול` (ארוך)
- `התחל את התהליך` (מעורפל)
- `שלח` (גנרי) — תגיד מה שולחים: `שמור פרופיל`
- `בוצע` (לא ברור מתי)

---

### Status / Live Data
**טון:** מספרים יבשים + יחידה. תמיד `mono` font.

✅ **טוב:**
```
GPS ON
2.34 ק"מ
45 שנ'
↑ צפון
ציון בטיחות: 85%
```

❌ **לא:**
- `GPS פעיל ועובד מעולה` (מילולי)
- `כשני קילומטרים בערך` (לא מדויק)

---

### Toasts / Confirmations
**טון:** קצר, חיובי, ענייני.

✅ **טוב:**
- `נשמר`
- `נמחק`
- `הקישור הועתק`

❌ **לא:**
- `הפעולה הסתיימה בהצלחה רבה!` (פורמלי + נלהב)
- `Done!` (גנרי + אנגלית)

---

### Confirmations / Destructive Actions
**טון:** ברור מה תהיה התוצאה. שאלה ישירה.

✅ **טוב:**
```
למחוק את המקלט הזה?
[כן, מחק] [ביטול]
```

❌ **לא:**
- `האם אתה בטוח?` (לא ברור על מה)
- `פעולה זו תמחק את המקלט. להמשיך?` (פורמלי)

---

## Hebrew / English Rules

### Default: Hebrew
כל UI מילה — עברית.

### English allowed only when:
1. **אקרונים מקובלים שאין להם תרגום טוב:** `SOS`, `GPS`, `OSRM`, `PWA`, `URL`, `API`, `OK`
2. **שמות מותג / טכנולוגיה:** `Firebase`, `Leaflet`, `Vercel`
3. **מספרים / יחידות בינלאומיות:** `2.4 km` ייתכן, אבל מעדיף `2.4 ק"מ`

### Don't:
- ❌ אל תערבב אנגלית באמצע משפט עברי בלי סיבה: "*Loading* את הנתונים" → "טוען נתונים"
- ❌ אל תכתוב "Click here" — תכתוב "לחץ"
- ❌ אל תשאיר placeholders באנגלית: `<input placeholder="Email">` → `placeholder="אימייל"`

---

## Forbidden Words / Phrases
**אסור בשום מקום באפליקציה:**
- "אופס" / "Oops" — לא רציני
- "מצטערים" / "סליחה" — לא מקצועי
- "מהר תיכנס למקלט!!!" — פאניקה
- "טוען..." בלי הקשר — תגיד מה טוען: "טוען מקלטים..."
- אימוג'י באמצע משפט פרוזה: "המקלט קרוב 🏃‍♂️💨"

---

## Allowed Emojis (sparingly)
**רק לסטטוס / categorization:**
- ✅ הצלחה / מוכן
- 🟡 אזהרה / חלקי
- 🔴 חירום / חסר
- 🛡 מקלט (אייקון רשמי)
- ↑ ↗ → ↘ ↓ ↙ ← ↖ — חצי כיוון

**לא בכותרות, לא בכפתורים, לא בפסקאות.**

## Tone Cheat Sheet

| מצב | טון | מילים שמתאימות | מילים שלא |
|------|-----|----------------|------------|
| חירום | דחוף, פעמי | אזעקה, היכנס, עכשיו, מקלט, X מטרים | אנא, בבקשה, אופציה |
| שגרה | יבש, ענייני | GPS, מוכן, סומן, פעיל | מצוין, נהדר, וואו |
| שגיאה | עובדתי | לא זמין, בדוק, חסר | אופס, מצטערים |
| empty | מדריך | אין X, [פעולה] | רוקן, כלום |
| כפתור | ציווי | תכנן, פתח, שמור | בבקשה לחץ, התחל |
| מספרים | מדויק | 2.34 ק"מ, 45 שנ' | בערך, כמעט |
