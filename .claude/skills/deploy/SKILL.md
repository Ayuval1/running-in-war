# Deploy — RunningInWar

1. ודא שאתה בתיקיית הפרויקט הראשית (לא worktree)
2. הרץ `git push` לפני deploy
3. הרץ `cd app && vercel --prod`
4. דווח על ה-URL שקיבלת

**חשוב:** תמיד מריץ מתוך `app/` — לא מה-root.
**Production:** https://running-in-war.vercel.app
**GitHub:** https://github.com/Ayuval1/running-in-war
