# Brain Index — המוח המשותף

## מה זה B-brain
המוח המשותף של כל הסוכנים. כאן נמצא כל הידע שצברנו — לא זהות המוצר (זה C-core), אלא **ידע שנצבר לאורך הדרך**.

סוכן שצריך מידע → בא לכאן. לא קוראים את כל הקבצים בתחילת שיחה — הולכים לקובץ הספציפי שצריך.

---

## מה שייך לכאן

| קטגוריה | דוגמאות |
|----------|---------|
| **מוצר** | מפרט, user stories, החלטות design |
| **מחקר שוק** | ניתוח מתחרים, benchmark, השוואות |
| **השראות עיצוביות** | references, moodboards, apps שאנחנו אוהבים |
| **אנליטיקס** | נתוני שימוש, conversion, retention |
| **ארכיטקטורה** | החלטות טכניות, schemas, API docs |
| **Roadmaps** | תוכנית עתידית, פיצ'רים מתוכננים |
| **ידע כללי** | כל מידע שכדאי לשמור לשימוש עתידי |

---

## קבצים קיימים

| קובץ | מה בו |
|------|--------|
| `product.md` | מפרט המוצר — users, purpose, principles, anti-references |

---

## איך להשתמש
- **סוכן צריך מידע על המוצר** → `product.md`
- **מוטי סיים מחקר** → שומר תוצאות ב-`O-output/Owner's Output/`, ואם מדובר בידע שיישמר לאורך זמן → מעביר לכאן
- **התקבלה החלטה ארכיטקטורית** → נרשמת כאן
