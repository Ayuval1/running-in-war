# ריצה בזמן מלחמה 🏃

**ה-Waze של מקלטי ישראל.**

אפליקציה שעוזרת לך להכיר את המקלטים באזור שלך — לפני שמשהו קורה.

---

## הרעיון

לא אפליקציית חירום — **אפליקציית הכנה**.

אתה פותח אותה בשקט, מסמן את המקלטים שאתה מכיר, ומכיר את הסביבה שלך מראש.
ובמידה ושכחת — כפתור ה-SOS האדום שם בשבילך.

> "אני יודע מה לעשות אם משהו יקרה."

---

## 3 עמודי התווך

| | פיצ'ר | תיאור |
|---|---|---|
| 🗺️ | **הכן מראש** | סמן מקלטים בשקט, לפני שמשהו קורה |
| 🚨 | **SOS בחירום** | כפתור אחד — חץ כיוון + זמן ריצה — ברור ומהיר |
| 🤝 | **קהילה בונה** | כל תושב מזין מקלטים, כל העיר מרוויחה |

---

## טכנולוגיות

- **Frontend:** React + Vite + Tailwind CSS
- **מפות:** Leaflet + OpenStreetMap
- **Backend:** Firebase (Auth + Firestore)
- **Deploy:** Vercel (PWA)

---

## התקנה

```bash
npm install
npm run dev
```

משתני סביבה נדרשים (Firebase):
```
VITE_FIREBASE_API_KEY
VITE_FIREBASE_AUTH_DOMAIN
VITE_FIREBASE_PROJECT_ID
VITE_FIREBASE_STORAGE_BUCKET
VITE_FIREBASE_MESSAGING_SENDER_ID
VITE_FIREBASE_APP_ID
```

---

## החזון

**אין אזרח בישראל שמגיע למקום ולא יודע איפה המקלט הקרוב אליו.**

כמו ש-Waze הפך כל נהג לשותף ברשת התנועה — כל תושב הופך לשותף ברשת המקלטים.
ככל שיותר אנשים משתמשים, כך מפת המקלטים מדויקת יותר.
