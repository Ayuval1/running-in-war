# Skill: explore-structure

**מי משתמש:** מוטי בלבד
**מתי:** כל בקשה לחקור מבנה קוד, תיקיות, קבצים, "מה יש ב-X", "איפה נמצא Y"

---

## Trigger
כל אחד מאלה → טען skill זה:
- "תחקור את מבנה..."
- "מה יש ב-[תיקיה/פרויקט]?"
- "איפה נמצא [קובץ/פונקציה/קומפוננטה]?"
- "תמפה את..."
- "תסרוק את..."
- JOHN שולח מוטי לסרוק פרויקט לפני גיוס סוכן

---

## Workflow

### שלב 1 — עץ ראשי
```
Glob("**/*", { path: "[root]", maxDepth: 2 })
```
מחזיר את כל הקבצים/תיקיות עד עומק 2. מספיק לתמונה כללית.

### שלב 2 — קרא קבצי מפתח
תמיד קרא (אם קיימים):
- `CLAUDE.md` — הוראות פרויקט
- `README.md` — תיאור כללי
- `A-agents/roster.md` — מי בצוות
- `T-tools/index.md` — כלים זמינים
- `M-memory/learning-log.md` — לקחים קיימים
- `package.json` / `vite.config.js` — stack (אם פרויקט קוד)

### שלב 3 — חיפוש ממוקד (אם יש target ספציפי)
```
Grep("[pattern]", { path: "[root]" })
```
לדוגמה: "איפה מוגדר useLocation?" → Grep("useLocation", { path: "app/src" })

### שלב 4 — top-level מהיר (אופציונלי)
```
Bash: ls [path]
```
לאימות ויזואלי מהיר.
